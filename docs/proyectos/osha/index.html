Descripcion

Este juego fue un ejercicio como parte del master de videojuegos y realidad virtual, en el que teniamos que hacer un plataformas en una semana. Sintiendo que tenia un buen entendimiento del motor decidi que no solo iba a hacer el codigo de un plataformas si no que decidi hacer todos los elementos del mismo en una semana, aparte de enfretarme a algunos retos que no habia hecho personalmente antes. Asi que hice tanto el diseño del juego, como visuales, como sonido, como musica, y la programación en una semana. Este proyecto lo hice usado Unity como motor, AseSprite para los sprites y para la musica y efectos de sonido use FL Studio
