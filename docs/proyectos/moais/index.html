Descripcion

Este curso fue el proyecto final de la clase de animacion del grado superior de Animacion 3D, Juegos y Entornos Interactivos de Cesur. En este proyecto aparte de animar algunas secciones, tambien ayude a texturizar la isla y mas importante ayude en la parte tecnica, ya que fui necesario para poder usar el sistema de agua de Unity, hacer codigo para que se meciesen la palmeras de manera sutil, y formular un "pipeline" para pasar todo facilmente de Blender a Unity. Este proyecto se hizo con Blender, Substance Painter y Unity. Si quieren saber del resto del equipo les invito a ver el corto y sus creditos.
